◆ getAIDeviceHandler()

uint16_t * getAIDeviceHandler ( void )

Retrieve the device handlers for AI (Analog Input) modules.

This API returns an array of handlers for AI modules. The first element in the array (index 0) contains the total number of handlers. The subsequent elements (index 1 to N) represent the actual handlers.

Parameters
void
Returns
uint16_t* A pointer to an array of handlers for AI modules.
  • Index 0: Number of handlers.
  • Index 1 to Index (N): DI module handlers.
Note
  • The maximum size of the array is 9, where index 0 contains the count of handlers.
