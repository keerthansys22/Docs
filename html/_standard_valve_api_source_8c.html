◆ setStandardValveOutputValue()

uint8_t setStandardValveOutputValue ( uint16_t pressurezone_handler,
uint8_t * station_data )

Set standard valve output values for the specified pressure zone.

Parameters
pressurezone_handlerThe handler ID for the target pressure zone.
station_dataPointer to a 6-byte array containing valve output statuses.
  • 1 byte contains 4 station statuses (2 bits per station: COIL A and COIL B).
  • Format: Byte 0: STA1–STA4 Byte 1: STA5–STA8 Byte 2: STA9–STA12 Byte 3: STA13–STA16 Byte 4: STA17–STA20 Byte 5: STA21–STA24
Returns
RET_OK on success, RET_ERROR on failure.
Note
The station_data pointer must reference exactly 6 valid bytes.
