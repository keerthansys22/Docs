RetStatus_T getDiagValveProcessData ( uint16_t pressurezone_handler,
uint16_t diagvalve_handler,
diagvalve_processinfo * diagvalve_processdata,
uint8_t * flag )

Retrieves diagnostic process data for a specified diagnostic valve.

This function fetches the diagnostic process data associated with a given diagnostic valve handler within a specified pressure zone.

Parameters
[in]pressurezone_handlerThe handler for the pressure zone.
[in]diagvalve_handlerThe handler for the diagnostic valve.
[out]diagvalve_processdataPointer to a structure where the retrieved process data will be stored.
[out]flagThis is give the status of any events being clear from plc [0x01] - Cleared from PLC [0x00] - Not cleared from PLC
Returns
RET_OK on success or error code
