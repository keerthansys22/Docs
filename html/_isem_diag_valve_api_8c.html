uint16_t isem_diag_handler,
isemdiag_processinfo * processdata,
uint8_t * flag )

Retrieves diagnostic process data for a specified ISEM diagnostic handler.

This function fetches the diagnostic process data associated with a given ISEM diagnostic handler within a specified pressure zone.

Parameters
[in]pressurezone_handlerThe handler for the pressure zone.
[in]isem_diag_handlerThe handler for the ISEM diagnostic module.
[out]processdataPointer to a structure where the retrieved process data will be stored.
[out]flagThis is give the status of any events being clear from plc [0x01] - Cleared from PLC [0x00] - Not cleared from PLC
Returns
RET_OK on success or error code
