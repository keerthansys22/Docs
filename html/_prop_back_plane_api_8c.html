uint16_t external_prop_handler,
prop_bckpln_processinfo * extrpropbck_processdata )

Retrieves diagnostic process data for a specified External Prop Backplane.

This function fetches the diagnostic process data associated with a given proportional station handler within a specified pressure zone.

Parameters
[in]extproppz_handlerThe handler for the External Prop PZ Zone handler
[in]external_prop_handlerThe handler for the External Prop Backplane.
[out]extrpropbck_processdataPointer to a structure where the retrieved process data will be stored.
Returns
uint8_t Returns RET_OK on success, or an error code on failure. Refer to e_error_code for possible error codes.
