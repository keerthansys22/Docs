This API is to clear the warning and fault bits of the modules @Note : This API can clear particular bit as well as the entire queue.

Parameters
[in]pressurezone_handlerThe handler for the pressure zone.
[in]device_handlerThe handler for the isem diag or diag valve
[in]flagFlag represents clear the entires warnings or particular bit . [0x01] - Clear all [0x02] -Clear particular bit
[in]warning_fault_flagThis flag represents warning event or fault event [0x01] - Warning [0x02] - Fault
[in]wfdataPointer to the buffer containing the data to be set.
