◆ FirmwareUpgradeStatus()

void FirmwareUpgradeStatus ( uint32_t * totalpackets,
uint32_t * packetsentsuccess,
uint8_t * errsts )

Get the current firmware upgrade status.

This function provides the total number of packets to be sent, the number of packets successfully transferred, and any error status.

Parameters
[out]totalpacketsPointer to store the total number of packets (including header and end packets).
[out]packetsentsuccessPointer to store the number of packets successfully sent.
[out]errstsPointer to store the error status.
Note
  • totalpackets will be set to size_var + 2.
  • packetsentsuccess will be set to the value of g_success_packets.

Example usage:

uint32_t total = 0, success = 0;
uint8_t err = 0;
FirmwareUpgradeStatus(&total, &success, &err);
printf("Total: %u, Success: %u\n", total, success);

Updates the firmware of the specified module.

This API initiates the firmware upgrade process for a given device.
