Retrieves diagnostic process data for a specified Internal Prop.

This function fetches the diagnostic process data associated with a given proportional station handler within a specified pressure zone.

Parameters
[in]pressurezone_handlerThe handler for the pressure zone.
[in]intprop_handlerThe handler for the Internal Prop.
[out]propstation_processdataPointer to a structure where the retrieved process data will be stored.
Returns
uint8_t Returns RET_OK on success, or an error code on failure. Refer to e_error_code for possible error codes.
