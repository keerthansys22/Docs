◆ ipc_receive_message()

uint8_t ipc_receive_message ( int fd,
s_ipcmsg_packet * ipc,
int timeout_ms )

Receive an IPC message from a remote endpoint with timeout support.

This function attempts to receive an IPC message from the specified file descriptor. The operation will block until a message is received or the specified timeout elapses.

Parameters
[in]fdFile descriptor to receive the message from.
[out]ipcPointer to the s_ipcmsg_packet structure to store the received message.
[in]timeout_msTimeout duration in milliseconds. If the timeout expires before a message is received, the function returns IPC_TIMEOUT.
Returns
  • IPC_RET_OK on successful reception.
  • IPC_RECEIVE_ERROR on failure.
  • IPC_TIMEOUT if the timeout period expires with no message received.
