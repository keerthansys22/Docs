Access(read or write ) the parameters for a specific instance and channel in an I/O module.

This function reads parameters from a specific IO modules

Parameters
instanceInstance of the I/O module (e.g., 1, 2, ...).
board_idBoard ID of the I/O module (e.g., DIO_M08_BOARD_ID, AI_M12_BOARD_ID, etc.).
chnlnumberChannel number within the specified instance and board.
parameter_idIdentifier of the parameter to access.
sizeSize of the data to be read in bytes.
valPointer to the buffer where the read data will be stored.
rdwrRead or write the value;
Returns
uint8_t Status code indicating the result of the operation:
  • RET_OK on success.
  • INVALID_PARAM if an invalid parameter is passed.
  • INVALID_BUFFER if the buffer pointer is invalid.
  • SPINLCK_TIMEOUT if the spinlock could not be acquired within the timeout period.
  • INVALID_BOARD_ID if the board ID is not recognized.
  • ‘INVALID_ACCESS’ for parametrs that does not have the persmision
