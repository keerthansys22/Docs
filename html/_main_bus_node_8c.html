◆ getMainBusMacAddress()

RetStatus_T getMainBusMacAddress ( uint16_t parameter_name,
uint8_t mac_address_value[6] )

Retrieves the MAC address of the specified main bus port.

This function reads the main bus service data from shared memory and retrieves the MAC address corresponding to the requested port. The MAC address is returned as a 6-byte value in the provided buffer.

Parameters
parameter_nameThe identifier for the port whose MAC address is requested. This can be one of:
  • MBN_MAC_ADDRESS_PORT1
  • MBN_MAC_ADDRESS_PORT2
  • MBN_MAC_ADDRESS_PORT3
mac_address_valuePointer to a buffer where the 6-byte MAC address will be stored. The caller must ensure that this buffer has at least 6 bytes of space.
Returns
RET_OK on success or error code
