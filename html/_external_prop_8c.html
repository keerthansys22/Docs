Retrieves process data for a specified External Prop.

This function fetches the diagnostic process data associated with a given proportional valve handler within a specified External Prop .

Parameters
[in]extprop_handlerThe handler for the External Prop
[out]pzxextprop_processinfoPointer to a structure where the retrieved process data will be stored.
Returns
uint8_t Returns RET_OK on success, or an error code on failure. Refer to e_error_code for possible error codes.
